📖 Grimório Mágico

E-commerce Temático de Poções e Feitiços

Sobre o Projeto

Sistema de e-commerce desenvolvido em Laravel com temática mágica/medieval, onde usuários podem comprar poções e feitiços que são armazenados em um grimório pessoal interativo. O projeto combinaFuncionalidades modernas de e-commerce com uma interface imersiva e criativa.

Funcionalidades Principais

Para Usuários:
  • Sistema completo de autenticação
  • Catálogo de produtos (Poções e Feitiços)
  • Carrinho de compras
  • Sistema de checkout
  • Grimório pessoal interativo com efeito flipbook
  • Perfil com upload de avatar
Para Administradores:
  • Painel administrativo protegido
  • CRUD completo de produtos
  • Upload de imagens
  • Controle de categorias e preços
  • Sistema de roles (user/admin)

Stack Tecnológica

Backend: PHP 8.2+ Laravel 12 SQLite Eloquent ORM
Frontend: Blade Templates Tailwind CSS Alpine.js jQuery + Turn.js

Destaques Técnicos

  • Padrão MVC: Arquitetura bem estruturada com Models, Controllers e Views
  • Sistema de Carrinho: Implementação usando sessões do Laravel
  • Upload de Imagens: Validação e armazenamento em storage público
  • Grimório Interativo: Integração criativa do Turn.js para experiência de livro virtual
  • Middleware Customizado: Controle de acesso baseado em roles
  • Relacionamentos Eloquent: One-to-Many entre User → Orders → OrderItems

Aprendizado

Este projeto me permitiu aprofundar conhecimentos em Laravel, padrões MVC, relacionamentos de banco de dados e integração de bibliotecas externas. Utilizei documentação oficial, tutoriais e ferramentas de IA para acelerar o Aprendizado de conceitos mais complexos.

Métricas do Projeto

5+ Controllers
15+ Views Blade
9 Migrations

Organizador Moodle & SUAP

Aplicação Desktop de Automação Acadêmica

Sobre o Projeto

Aplicação desktop desenvolvida em Python que automatiza a coleta e organização de informações acadêmicas do IFSul. O sistema integra dados do Moodle (plataforma de ensino) e do SUAP (Sistema Unificado de Administração Pública) em uma interface gráfica moderna e intuitiva.

Funcionalidades Principais

  • Web Scraping Automático: Coleta de tarefas e atividades do Moodle usando Selenium
  • Integração SUAP: Extração automática de dados de frequência e faltas
  • Gerenciamento de Tarefas: Criação manual de tarefas personalizadas
  • Interface Dark Mode: Design profissional com sistema de abas
  • Sistema de Cache: Persistência local em JSON para carregamento rápido
  • Sincronização Background: Threading para não travar a interface

Stack Tecnológica

Python 3 Tkinter Selenium ChromeDriver Threading JSON PyInstaller

Destaques Técnicos

  • Arquitetura Modular: Separação clara entre scraping, GUI e lógica de negócio
  • Web Scraping Robusto: Navegação headless com tratamento de exceções
  • Threading Pattern: Comunicação thread-safe com Queue
  • Parsing Inteligente: Extração de datas em português
  • Build Standalone: Executável .exe com PyInstaller

Desenvolvimento

Este projeto resolveu um problema real do meu dia a dia acadêmico. Aprendi web scraping, programação assíncrona e criação de interfaces desktop. Utilizei documentação, tutoriais e ferramentas de apoio para compreender conceitos mais complexos de threading e automação.

💬 Vox Arcanum

Aplicativo de Chat em Tempo Real

Sobre o Projeto

Aplicativo mobile multiplataforma de mensagens instantâneas desenvolvido em Flutter, com temática mística/arcana. O app oferece chat em tempo real comFuncionalidades avançadas de presença online, notificações push, sistema de atualização automática e proteção de privacidade.

Funcionalidades Principais

  • Chat em Tempo Real: Conversas privadas e chat geral com sincronização Firestore
  • Sistema de Presença: Detecção automática online/offline com Realtime Database
  • Notificações Push: Firebase Cloud Messaging integrado
  • Atualização OTA: Sistema próprio de download e instalação de updates
  • Proteção de Privacidade: Screenshot protection em conversas
  • Perfil Customizável: Upload de foto e personalização de nome

Stack Tecnológica

Flutter Dart Firebase Auth Firestore Realtime DB FCM Firebase Storage

Destaques Técnicos

  • Dual Database: Uso inteligente de Firestore + Realtime DB para otimizar custos
  • Paginação Eficiente: Carregamento incremental de mensagens (20 por vez)
  • Sistema OTA Próprio: Atualização sem depender de lojas de aplicativos
  • Arquitetura em Camadas: Separação clara entre Services e Screens
  • Gerenciamento de Estado: StreamBuilder para dados em tempo real

Desenvolvimento

Projeto complexo que me ensinou integração completa com Firebase, gerenciamento de estado em Flutter e implementação de features avançadas como sistema de presença e atualização OTA. Estudei documentação oficial e utilizei recursos de Aprendizado para dominar conceitos de programação assíncrona e streams.

Versão atual: 1.3.0 | Plataformas: Android, iOS, Web, Desktop
